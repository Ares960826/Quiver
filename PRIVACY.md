# Privacy

Quiver is local-first. The Community Preview does not require an account and
does not upload prompt contents.

- Prompt files remain in the folder selected by the user.
- Usage history is stored only in a disposable local cache.
- Quiver does not sell access to user data.
- This GitHub repository receives only the information a user chooses to post
  in an issue or discussion.

When reporting a problem, remove private prompt text, personal paths and other
sensitive information from screenshots and logs.
