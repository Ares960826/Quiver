# Quiver visual showcase

[English](SHOWCASE.md) · [简体中文](SHOWCASE.zh-CN.md) · [한국어](SHOWCASE.ko.md)

Quiver keeps complete prompts and reusable templates in one local, searchable
library. These examples show how different languages, slot types and outputs
fit together without turning the repository homepage into a screenshot gallery.
