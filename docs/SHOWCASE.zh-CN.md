# Quiver 图文展示

[English](SHOWCASE.md) · [简体中文](SHOWCASE.zh-CN.md) · [한국어](SHOWCASE.ko.md)

Quiver 把完整 Prompt 与可复用模板放在同一个本地搜索库中。下面用不同语言、
空格子类型和输出方式展示它们如何配合，同时让仓库主页保持简洁。

<p align="center">
  <img src="../assets/showcase/arrow-library-multilingual.png" width="650" alt="同一个 Quiver 箭袋中收纳了多语言的完整 Prompt 与可复用模板">
  <br><sub><strong>一个箭袋，多种 Arrow。</strong>完整 Prompt、可填写模板和多语言内容都在同一个搜索库中，不必拆成几套管理方式。</sub>
</p>
