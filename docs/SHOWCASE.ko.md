# Quiver 이미지 쇼케이스

[English](SHOWCASE.md) · [简体中文](SHOWCASE.zh-CN.md) · [한국어](SHOWCASE.ko.md)

Quiver는 완성된 프롬프트와 재사용 가능한 템플릿을 하나의 로컬 검색 라이브러리에
보관합니다. 아래 예시는 홈페이지를 길게 만들지 않으면서 여러 언어, 슬롯 유형,
출력 방식이 함께 작동하는 모습을 보여 줍니다.

<p align="center">
  <img src="../assets/showcase/arrow-library-multilingual.png" width="650" alt="여러 언어의 완성형 프롬프트와 재사용 템플릿이 함께 있는 Quiver 라이브러리">
  <br><sub><strong>하나의 화살통, 여러 종류의 Arrow.</strong> 완성형 프롬프트, 입력형 템플릿, 다국어 자료를 같은 검색 라이브러리에 보관합니다.</sub>
</p>
