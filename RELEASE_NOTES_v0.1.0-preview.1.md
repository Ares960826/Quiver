# Quiver v0.1.0 Community Preview 1

The first public preview of Quiver: a local-first prompt manager and reusable
template launcher for macOS.

## Included

- Global launcher with a customizable hotkey
- Search across titles, tags, groups and prompt bodies
- Complete prompts and reusable templates in one library
- Text, choice and clipboard-powered slots with live preview
- English and Simplified Chinese app interfaces
- Five themes, including colour-safe and high-contrast options

## Download

- `Quiver_0.1.0_aarch64.dmg` — Apple Silicon Macs
- `Quiver_0.1.0_x64.dmg` — Intel Macs
- `SHA256SUMS.txt` — SHA-256 checksums

## macOS security notice

This preview is ad-hoc signed and is not notarized by Apple. After moving
Quiver to Applications, try to open it once. If macOS blocks it, open
**System Settings → Privacy & Security**, choose **Open Anyway**, and confirm
**Open**.

Please report reproducible problems through GitHub Issues after removing any
private prompt text, personal paths or sensitive screenshots.
